About This Project

This data visualization project aims to identify the effectiveness of Missing Glyph Visualization tool, specifically through its linear layout on representing the missingness patterns on multivariate datasets with different sizes (number of variables). I highly suggest you read the README documentation first before accessing the visualization tool to make things more clear.

Features

Each element and feature in the visualisation is explained in my README documentation. In case you forgot about them, there are several things that you must know before you start exploring: -

  • Missing Data Pattern Analysis & Statistical Insight
    • Each rectangular container represents a single variable in a dataset with its missingness patterns.
    • Amount Missing (AM) is represented by the blue bar (located on the right side of the container)
    • Joint Missingness (JM) is represented by the red bar (located on the right side of the container) and the red arch (connected between containers that share missing values with the selected variable)
    • Conditional Missingness (CM) is represented by the distribution of recorded values (red histogram) that is located on the left side of the container. Provided that the selected variable contain missing values, the missing values in it are considered as conditional if its corresponds to clear patterns of distributions on the recorderd values (red histogram) for other variables.
    • Gray histogram (positioned on the left side of the container) represents all the recorded values of the variable.
  • Interactive Data Visualization & Easy-to-use Interface
    • Zooming in and zooming out using your laptopâ€™s touchpad or your somputer's mouse.
    • Hovering over a bar, a histogram bar and an arch , will reveal a tooltip that show its values and extra information like category and range.
    • Resetting the visualization to its default state can be done by clicking on the 'R' key.
